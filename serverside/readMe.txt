to use this script, you need:

- MongoDB installed on the same computer that you intend to run the script on.
	- https://docs.mongodb.com/manual/installation/

- the TwitterAPI Python library
	- https://github.com/geduldig/TwitterAPI

- API keys from Twitter.
	-https://dev.twitter.com/ads/tutorials/getting-started

There are four and each one needs to be entered in the places marked 'XXXX' in the script.
